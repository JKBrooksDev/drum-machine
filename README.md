# Drum Machine

A responsive, interactive drum machine built with React and Vite. Users can trigger sounds using on‑screen pads or keyboard keys, with real‑time display updates and smooth audio playback.

## Features
- Clickable drum pads with instant audio playback
- Keyboard support for triggering sounds
- Dynamic display showing the active sound
- Clean, responsive UI for desktop and mobile
- Built with React + Vite for fast performance

## Tech Stack
- React
- Vite
- JavaScript (ES6+)
- CSS
